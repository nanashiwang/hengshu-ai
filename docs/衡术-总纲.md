## 1. 产品定位

**衡术是一个中心化运营的 AI Skill 平台，"决策层 + 履约层"一体机**：
- **决策层**：中立评测 + 省钱路由——用真实运行数据告诉用户"这个 Skill 在哪个模型上真省真稳"，并自动帮你选当前实测最省够用的模型。
- **履约层**：作者自有的 **New API 中转**——模型调用真正发生的地方，平台从 margin 赚钱。
- **算力 = 用户自带 Key（BYOK），默认走自己中转**；平台不承担现金算力成本，从中转 margin 变现。
- **公网多用户中心化**（非自部署）；单一中心 → 兼容报告/术值榜/求术全站聚合，PT 网络效应成立，信任模型为中央签名。

## 2. 护城河与竞争力（战略核心）

**诚实前提**：单纯"攒兼容数据"不是护城河——结论低密度可爬、评分不奖励样本量、网关在数据上游、数据会腐坏、看一眼就走。**护城河必须挪到"长在真实 BYOK 流量上、且创造切换成本"的地方。**

**加固后的 5 层护城河**（均以已有代码为基座；标注"目标态"的部分尚未实现，勿当现状引用）：

| 层 | 是什么 | 为何抄不走 |
|---|---|---|
| **0 活体数据** | `compat.ts` 聚合层：现状=modelName 单级分组+时间衰减(halfLife=30天)+置信度折减；**目标态(阶段4)**=(modelName,modelVersion) 二级分组+漂移曲线 | 竞品抓走"今天"，抓不走连续过去与跨版本漂移曲线；腐坏反成壁垒 |
| **1 省钱路由** ⭐ | **#15前半已通电(2026-07-02)**：`recomputeRoutePolicy` 由真实回流回写 `routePolicy.dataDriven`，`selectModel` 优先数据驱动、漂移随回流自刷新。**现状：数据驱动路由已生效**；⏳ console"省钱回执"UI 与 per-user 个人化(后半) | 抄"看表"易，抄"真实回流驱动的自动降本路由"须先有等量真实运行 |
| **2 私人台账** ⭐ | 激活 `SkillRuns`（已存 input/output）做"我的运行台账 + 换模型一键重跑" | 用户**自己的**数据，非公共平均；迁移成本随使用线性上升；堵死 BYOK"查完去网关直连"漏点 |
| **3 任务可靠性** | 聚合键从 modelName 下移到 profile(skill×输入档×errorType)；代际回归告警 | 模型趋同后不消失反更值钱（prompt 工程质量差异不趋同） |
| **4 可验签基准** | verified + ed25519 升级为第三方达标证书；黄金样例逐条打分 | 网关自评自证有利益冲突，中立性结构上是你的 |

**⭐ 最锋利竞争力（复购与切换成本唯一来源）= 第1层×第2层**：
> **"绑你的 Key 后，用你自己历史 prompt 的真实回流，持续替你选最省的模型跑、漂移自动换路由、并累计告诉你这月省了多少钱。"**
> 个人化 + 每周涨可推送 + 离开就丢台账——省的钱沉淀在平台账本而非用户脑子里。

### 2.1 六维红队复核定论（2026-07-02：6 类候选 × 竞品红队，全部按最狠打法攻击）
**残酷事实：没有任何一层"今天"就成立**（评级：切换成本=C，数据/网络效应/信任/成本结构/标准=D）。但每层的成立开关都已定位，多为天级工作量。核心定论：

1. **数据不是独立护城河，是第1×2层的派生物**。壁垒深度公式 = **独占流入速率 × 半衰期**，不是累计条数。规模阈值有算术：全站需 ~77 次真实运行/天才能让 25 Skill×4 模型的兼容矩阵整体"满置信活着"——低于此值护城河深度记零，不对外宣传。"网关没有 Skill 语义"防线偏弱（response_format/json_schema 可让网关推出等价 formatValid），真独占的只有：**私人台账（ownerOrAdmin 天然不可爬）+ 真实用户自然流量分布 + Skill 语义连接键**。
2. **两面承重墙（红队打不穿的仅有位置）**：① **个体切换成本**——SkillRuns 私人台账 × 省钱回执（累计省钱额是唯一**不随 30 天半衰期归零**的单调增长资产，留存钩子全押它）；不需要网络密度，solo+¥500/月唯一养得起的形态。② **跨中转中立元台账**——国内 New API 中转极度碎片化，没有任何单一网关看得到用户跨中转的全量调用；衡术定位为"跨中转的中立元台账/元路由"是**唯一对"网关上游超集"攻击免疫**的结构位置。
3. **counter-positioning 只对"自有模型利益"平台成立**（Coze/豆包、百度系抄真诚中立=自噬），**对中立聚合网关不成立**（OpenRouter 本就中立+有 :floor 价格路由，AWS Cost Explorer 证明"教客户省钱换留存"不是禁区）。低固定成本=氧气不是墙：它买的是熬到第1×2层长成的 12-24 个月时间。
4. **信任层单独不成立，只能当第0-2层的"信用放大器"**，且必须机器强制而非口号：四面墙目前 0/4 落地——"历史长度"因子从落地日才开始计时，**每晚一天轨迹短一天**。先自曝 margin 政策（披露在先=透明，被揭在后=丑闻）；SkillRuns 明文 input/output 是埋在第2层的**信任毁灭开关**（一次泄漏全层陪葬）。
5. **冻结两个叙事**：❄️"推 Skill Spec 事实标准"（solo 无分发杠杆+大厂格式收敛窗口期，最差时点；改做 import 适配器搭 Claude Skills/GPTs 便车，fork 谱系按"变异-表现差分数据采集器"立项）；❄️"网络效应"（唯一半通电的数据回路电流≈运营者本人；冷启动期不当卖点）。PT 邀请制**分层收缩**：只锁"挣术值/verified 计分/兑换"经济写路径，读+限额试跑放开流量（online 权重 0.3+封顶 3 条+6a 扣费已兜底）。

### 2.2 第二轮非常规候选红队定论（2026-07-02：合规/中文垂直/中转联盟/PT身份/失败知识库/B2B 采购）
**三个 B 级幸存者（全部是焊在承重墙上的钢筋，非第三面墙）：**
1. **中文×国产模型评测垂直（B）**——英文界(LMArena/AA)结构性不做、国产厂商自评=自噬、学术榜(SuperCLUE)不做任务级中文风格实测（公文体/小红书体/成语/¥千字真实分词成本）。是墙①②共同短板"获客"的引擎+信用钢筋。抄不走的只有：跨版本漂移时序、私有 held-out 套件+rubric、失败案例回路——价值曲线前 6 个月≈0，窗口期 6-18 个月（中立中转站入场前）。建设与既定必建项高度重叠（ModelPriceSnapshot=§3.3墙3、签名快照=6j、周榜页=#12 modelrank），增量仅 3-5 天 rubric 人肉+每月 ¥60-160 跑测费。**验收线：8-12 周"周报→注册→绑Key"转化不达标即降级月更**。
2. **失败知识库/负知识（B）**——没人系统攒"什么在哪个模型上怎么坏+怎么修"；单条确认案例即"救命"价值（不需统计显著性，匹配低流量现实）；SEO 长尾（"XX模型 json 截断"）竞争密度≈0，是 ¥500/月下唯一高意图获客渠道。喂路由后升维"成本×可靠性最优"（OpenRouter :floor 纯价格路由做不到）。⚠️ **水龙头当前关死**：skillRunner 6b 块失败运行零回流（见 6m）。防 AI 搜索零点击抽取：公开页只给症状+根因摘要，修复模板+复验工具锁登录后。
3. **PT 社区身份（B）**——机理修正：PT 活二十年靠的是"身份系统包裹稀缺品"，稀缺品在衡术=两面墙，身份只是配给层+放大层。**身份必须焊在省钱回执上而非归属感上**（这个人群"忠于省钱不忠于站"）；名号绑可验证真实数据+ed25519 快照。只做"产品功能"（个人主页/名号/创始编号/邀请谱系，5-7 天一次性），**绝不做论坛/群聊/广场**（UGC 先审后发义务+solo 时间黑洞双禁区）。创始编号 #001-100 每晚一天少一天，0.5 天先落。
**一张 C 级期权：中转联盟 probe-first**——把"中转抄你"反转为分发的唯一路径；但徽章无信誉时 BD 必碰壁 → 先用黄金样例探针跑"中转体检榜"（签名+外锚），静默 3 个月看 inbound：≥3 家主动求徽章则推进，零则永久降级为墙②自产数据源。防 Dieselgate 探针欺骗从第一天设计（随机化改写/普通用户账号/金丝雀题）。
**两个明确不做：** ❌ 合规当护城河（对有钱对手零阻挡；正确记账=经营许可证而非竞争壁垒，触发器=recharge 上线时点）；❌ B2B 合规采购（有合规痛点的企业买云厂商 MaaS，用中转的企业恰是绕开采购的人；且合规聚光灯会烧毁墙②生态位——唯一可能负外部性的候选。两年后以"团队版台账+国产 BYOK 通道"薄片重启，触发器=月自然企业询单≥3）。
**🚨 红队新发现的自爆装置（升级为 6l，与 6a 同改动点）**：当前 `runSkill` 默认 fallback=claude-haiku（境外未备案）+ 无 BYOK 静默回退平台 Key 代理——与 2026 年被整治的中转站同一法律位置（服务提供者定性）。72 小时架构切割：默认/fallback 链换已备案国产模型，境外模型仅 BYOK 显式配置可用。

## 3. 经济模型（变现，且反哺护城河）

### 3.1 三币单向漏斗
```
人民币 ──充值──▶ credit(算力燃料) ──跑模型──▶ 蒸发
贡献行为 ──挣──▶ 术值 ──受控兑换──▶ credit
                     术值/credit 永不反向变现金 ❌（保命红线：防资金盘/赌博）
```

| 币 | 角色 | 护栏 |
|---|---|---|
| CNY | 唯一入金口（New API 兑换码充值） | 衡术不碰现金；无任何→CNY 出口，代码硬禁止 |
| credit | 唯一花算力的币(1credit=¥0.01零售) | 物理映射为受限子令牌（低价分组+配额熔断+TTL）；free 档不产术值/不可兑换转让→白嫖上限=进货成本，套现≈0 |
| 术值 | 贡献信誉分（不可买只能挣） | 高价值行为**被采纳才发分**；不可转赠；永不进随机池 |

**核心不变量（结构性防亏）**：术值兑换池 = **α × 当月已实现毛利**(α=0.30) → 先赚到才有得兑，**永不亏 margin**，最坏少赚 30%。

### 3.2 抽奖的最终结论：**不做纯抽奖，做"满额返利宝箱"**
纯抽奖/开箱踩两条红线（反不正当竞争法有奖销售 + 变相赌博）。改：**充值满额 100% 必得**确定返利(6% 公示)，**随机只决定返利形态**（都是额度券，用户永不空手）——类"第二杯半价+随机送小料"。术值绝不进随机池。⚠️ 上线前走一次律师。

### 3.3 中立与变现共存：四面墙（护城河不被变现稀释）
1. **数学隔离**：`modelrank.ts` 排名函数入参**无 margin/isOurs 字段** + CI `grep` 断言（人事后也改不动）。
2. **记账隔离**：benchmark 按成本价记账(margin=0)；履约 SkillRuns 不进榜分。
3. **成本真值隔离**：用公开官方报价快照(`ModelPriceSnapshot`)非平台可编辑价；双列"官方价 vs 我到手价"，更贵必红标"BYOK更省"。
4. **履约隔离**：run 响应强制回传 `cheaperViaByok`，更省走 BYOK 不得隐藏。

### 3.4 New API 对接
1 个 New API 账号 = 全体用户，靠**子令牌**隔离（`token_name="hs_<userId>"`）。**已定架构（优于原稿）**：权威 credit 账本在本平台 `credit-logs`（原子事务+幂等），网关配额同步是提交后的**副作用**（失败不影响账本，靠对账 worker 补偿）——而非原稿"先扣术值→PUT /api/token→失败回滚"的跨系统事务。注册发受限子Key(60秒真跑)；cron 拉 `/api/log` 用量真值回流喂护城河(剥离明文)并回填当月毛利。

## 5. 统一落地路线（唯一开发顺序）

> 合并了原部署/成品/护城河/经济四份的所有优先级。按阶段推进，每阶段独立冒烟。

### 🚨 阶段 0 · 立即止血 + 护城河地基（本周，多为 0.5-2 天/项，不依赖外部决策）
1. **修 register 术值漏洞** ✅限速已做(2026-07-02，对抗审查确认正确无回归)：register 改为依赖规则表(不传 points)；invite/skill_version_update 加 `dailyLimit=3`。⚠️ **残余(未根除)**：① 自邀刷分——invitee≠inviter 使 `selfActionExcluded` 失效，需 #6 设备/IP 去重才能堵死。
